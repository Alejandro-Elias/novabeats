# Changelog

All notable changes to this project will be documented in this file.

This format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/)
and this project adheres to Semantic Versioning.

---

## [1.2.4] - 2026-06-27
### Added
- Updated MIT license in `package.json`.

### Fixed
- Fixed incorrect display of track duration.
- Updated project documentation in `README.md`.

---

## [1.2.3] - 2026-06-25
### Added
- Integrated `audioMotion-analyzer` visualizer.

---

## [1.2.2] - 2026-06-23
### Added
- Animation when switching between compact and normal modes.
- Playlist saving using the `Enter` key.

### Fixed
- Fixed shuffle button behavior.

---

## [1.2.1] - 2026-06-22
### Added
- Compact mode implementation.

---

## [1.2.0] - 2026-06-18
### Added
- Flatpak packaging configuration support.

---

## [1.1.4] - 2026-06-18
### Changed
- Improved performance when loading files and folders.

---

## [1.1.3] - 2026-06-17
### Fixed
- Fixed issue when adding files to an empty playlist.

---

## [1.1.2] - 2026-06-17
### Changed
- Improved performance for `next` action and playlist selection.
- Optimized metadata handling.

---

## [1.1.0] - 2026-06-15
### Added
- Playlist view for selected tracks.
- Option to delete playlists.
- Named playlist saving support.

---

## [1.0.5] - 2026-06-13
### Changed
- Switched track selection logic from index-based to ID-based system.

### Fixed
- Fixed issues with `next` and `previous` in shuffle mode.
- Fixed `repeat-all` bug.
- Allowed adding items to playlists correctly.

---

## Previous versions
- Versions `1.0.1` to `1.0.4` include early playback improvements, file selection support, and initial UX refinements.
